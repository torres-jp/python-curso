# 🐍 Curso de Python: Desde Básicos hasta Web Scraping

¡Bienvenido a este repositorio! Este es un curso completo de Python diseñado para llevarte desde los conceptos fundamentales hasta técnicas avanzadas como el web scraping. A lo largo del curso, encontrarás explicaciones detalladas, ejemplos prácticos y una serie de ejercicios para reforzar lo aprendido.

## 📚 Descripción

Este curso está estructurado para principiantes absolutos y personas con conocimientos básicos que desean profundizar en Python. Cubriremos desde la sintaxis básica, estructuras de datos, programación orientada a objetos, hasta el manejo de archivos, APIs y técnicas de web scraping para extraer datos de la web.

Cada módulo incluye:
- **Teoría explicada**: Conceptos claros y concisos.
- **Ejemplos de código**: Fragmentos ejecutables para ilustrar los temas.
- **Ejercicios prácticos**: Desafíos para aplicar lo aprendido y consolidar conocimientos.

## 🚀 Estructura del Curso

### 📁 Módulos Disponibles

- **01_basics/**: Fundamentos de Python
  - Variables y tipos de datos
  - Operadores y expresiones
  - Estructuras de control (if, loops)
  - Funciones básicas

*(Próximos módulos en desarrollo)*
- **02_data_structures/**: Estructuras de datos avanzadas
- **03_oop/**: Programación Orientada a Objetos
- **04_files/**: Manejo de archivos y excepciones
- **05_apis/**: Trabajo con APIs y JSON
- **06_web_scraping/**: Extracción de datos web

### 🎯 Objetivos de Aprendizaje

Al finalizar este curso, serás capaz de:
- Escribir programas Python eficientes y legibles
- Manipular datos de diversas fuentes
- Crear aplicaciones web simples
- Extraer y procesar datos de sitios web
- Resolver problemas reales con Python
